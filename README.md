产品种类涵盖高端低压注塑材料、BGA芯片封装材料、PCB板防护涂层、特种器件粘接和密封材料等。

### 📌 核心产品
- **低压注塑聚酰胺树脂**：低压注塑材料，是一类生物基特种注塑材料，非常环保并且可回收，兼具注塑性和高流淌性，它的应用领域介于普通注塑和化学灌封之间，特别适合于对温度、压力敏感的电子元器件和线束。
- **低压注塑聚酰胺树脂**：熔融粘度低，流淌性、浸润；符合欧盟RoHS、Reach要求，满足HF/VOC要求；防水、防潮、耐腐蚀、耐盐雾和电绝缘性能；固化速度快，适宜的粘接性；阻燃符合 UL94 V-0标准
- **胶黏剂**：底部填充胶（芯片封装）；双组分聚氨酯灌封（PU）；单组分环氧灌封；双组分环氧灌封
